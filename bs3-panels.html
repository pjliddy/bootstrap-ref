Bootstrap Reference - Panels

Panels

Basic Panel

A panel in bootstrap is a bordered box with some padding around its content.

Panels are created by adding the .panel class to a <div> and the .panel-body class to the <div> containing content inside the panel.

The .panel-default class is used to style the color of the panel.

Panel Heading and Footer

The .panel-heading class adds a heading to the panel.

The .panel-footer class adds a footer to the panel.

Panel Groups

Panels can be grouped together by wrapping them in a <div> with the .panel-group class.

The .panel-group class clears the bottom-margin of each panel.

Panels with Contextual Classes

Panels can be colored using contextual classes:

  • .panel-default
  • .panel-primary
  • .panel-success
  • .panel-info
  • .panel-warning
  • .panel-danger
