Fixed & Fluid Layouts

Fixed Layout

The basic grid system inside a .container div starts out stacked on extra small devices (phones)), switching to horizontal on medium/large devices.

Below is a basic "stacked-to-horizontal" two-column layout. It lays out content in two equal columns on all screens, except for small screens, where it will automatically stack as a single column, full-width layout:

The parent container sets its width according to the breakpoints defined by Bootstrap: 750px, 970px, and 1170px. Column width is a set proportion of this container width.

Layout Size Classes

Using Size Classes

In the grid below, the two columns are set to one-third and two-thirds of the container width on medium displays and higher using the .col-md-4 and .col-md-8 classes.

For small displays (>= 768), the layout switches to two equal width columns using the .col-md-8 class.

On extra small displays (< 768), the layout switches to a single, stacked column by default.

First Column
.col-sm-6 | .col-md-4
Second Column
.col-sm-6 | .col-md-8

Default Small Screen Layouts

If no .col-sm-* classes are used to specify layout behavior at smaller screen sizes, the layout will default to a single, stacked column below the lowest specified breakpoint (<= 992 px).

In the grid below, the two columns use the .col-md-6 class to define a layout with two equal columns on medium and large screen sizes. On small (tablet) and extra small (mobile) devices, the layout is a single column.

If .col-lg-* classes are the only ones used, the layout will default to single, stacked column below a screen width of 1200 px.

.col-md-6
Second Column
col-md-6

Large Screen Layouts

Large devices have a screen width of 1200 px or above. Different layout size classes can be applied for each screen size.

In the example below, the proportion of column widths changes with each defined breakpoint.

Second Column

Clear Floats

If content in columns will be of unequal height and produce uneven wrapping, add the .clearfix and .visible-size classes to a <div> after the correct number of columns for a row.

Offset Columns

Columns can be shifted a specifcied number of columns to the right using .col-md-offset-* classes.

These classes increase the left margin of a column by * columns. For example, .col-md-offset-4 moves .col-md-4 over four columns.

Offsets from smaller grid classes can be overridded with the .col-*-offset-0 classes.

.col-md-3
