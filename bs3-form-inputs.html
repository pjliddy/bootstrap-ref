Input

Bootstrap supports the HTML5 input types:

  • text
  • password
  • date
  • datetime
  • datetime-local
  • month
  • time
  • week
  • number
  • email
  • url
  • search
  • tel
  • color

Inputs won't be properly styled if their type is not declared.

Form Control States

Input Focus

When the input comes into focus (when user clicks on it), the outline of the input is replaced with a box-shadow.

Disabled Input

To disable an input, add the disabled attribute to an input field.

Disabled Fieldsets

To disable all inputs in a fieldset, add the disabled attribute to the fieldset element.

Read-Only Input

To make an input read-only, add the readonly attribute to an input field.

What's the difference between disabled and read-only?

Validation States

Add the .has-warning, .has-error, or .has-success class to the parent .form-group element of an input to use Bootstrap's validation styles for error, warning, and success messages.

To show feedback icons:

  • Add the .has-feedback class to the parent .form-group element
  • Add the .form-control-feedback class to an icon <span>in the parent .form-group.

Hidden Labels

Add the .sr-only class to a label to make it visible only to screen readers.

Sizing Inputs

Input Sizes

To set the size of an input element, add the .input-sm or .input-lg class to an <input> element.

Input Size on Horizontal Forms

To set the size of form controls and their labels on a horizontal form, add the .form-group-* class to the .form-group element.

Column Sizing

To assign different widths to input elements, add one of the column size classes (.col-xs-*) to the input's parent <div>.
