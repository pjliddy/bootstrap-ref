Bootstrap Reference - Page Headers & Jumbotron

Page Headers

Page Header

A page header is like a section divider. The .page-header class on a <div> wrapped around a heading adds a horizontal line under the heading (+ adds some extra space around the element)

Example Page Header

Jumbotron

A jumbotron is a big box for calling extra attention to some special content or information.

By default, a jumbotron is displayed as a grey box with rounded corners. It also enlarges the font sizes of the text inside it.

Nearly any valid HTML, including other Bootstrap elements/classes, can be used inside a jumbotron.

Inside of a Container

Applying the .jumbotron class to a <div> inside of a container will not extend to the edges of the screen.

A Jumbotron inside of a container will not extend to the edges of the screen.
