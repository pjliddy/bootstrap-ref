Button Size Classes

There are four button size classes in Bootstrap: .btn-lg, .btn-md, .btn-sm, .btn-xs

Button Style Classes

Bootstrap provides a default button class, which is usually used with one of the standard style classes.

Bootstrap provides seven styles of button classes: .btn-default, .btn-primary, .btn-success, .btn-info, .btn-warning, .btn-danger, .btn-link
