Dropdowns

Basic Dropdown

A dropdown is a toggleable menu that allows the user to choose one value from a predefined list.

A dropdown is created by applying the .dropdown class to a <div> containing a <ul> element.

The .dropdown-menu class is applied to the <ul> to create the menu.

The .caret class creates a caret arrow icon.

The dropdown is activated by a button or a link with a class of .dropdown-toggle and the data-toggle="dropdown" attribute.

Accessibility

To help improve accessibility for people using screen readers, include the following role and aria-* attributes, when creating a dropdown menu:

<div class="dropdown">
   <button class="btn btn-default dropdown-toggle" type="button" id="menu1" data-toggle="dropdown">
        Accessiblity <span class="caret"></span>
   </button>
   <ul class="dropdown-menu" role="menu" aria-labelledby="menu1">
       <li role="presentation"><a role="menuitem" href="#">First Menu Item</a></li>
       <li role="presentation"><a role="menuitem" href="#">Second Menu Item</a></li>
       <li role="presentation"><a role="menuitem" href="#">Third Menu Item</a></li>
       <li role="presentation" class="divider"></li>
       <li role="presentation"><a role="menuitem" href="#">Fourth Menu Item</a></li>
   </ul>
</div>

Bootstrap JS Dropdown

Dropdowns Using JavaScript

Dropdowns can be triggered manually with a JavaScript call to $('.dropdown-toggle').dropdown(); from a link or a button element

Note: The data-toggle="dropdown" attribute on the .dropdown-toggleis required even if calling the dropdown() method using JavaScript.

Events

The show.bs.dropdown event is triggered when the dropdown is about to be shown.

The shown.bs.dropdown event is triggered when the dropdown is fully shown (after CSS transitions have completed).

The hide.bs.dropdown event is triggered when the dropdown is about to be hidden.

The hidden.bs.dropdown event is triggered when the dropdown is fully hidden (after CSS transitions have completed).
