Bootstrap Reference - Glyphicons

Glyphicons

Glyphicon Class

Bootstrap includes 260 glyphs from the Glyphicon Halflings set

The .glyphicon class is added to span, along with a .glyphicon-name class:

<span class="glyphicon glyphicon-info-sign"></span>
