Bootstrap Reference - Grid System

Bootstrap Grid System

Bootstrap's grid system is responsive, so columns will re-arrange themselves depending on the screen size.

Bootstrap's grid system allows up to 12 columns across the page. Columns can be grouped together to create wider columns.

Grid Classes

The Bootstrap grid system has four classes that can be combined to create flexible layouts:

  • .col-xs-* for phones < 768 px (container width = auto)
  • .col-sm-* for tablets >= 768 px (container width = 750 px)
  • .col-md-* for desktops >= 992 px (container width = 970 px)
  • .col-lg-* for larger desktops >= 1200 px (container width = 1170 px)

Each class scales up, so if the widths for xs and sm are the same, you only need to specify xs.

Basic Structure of a Bootstrap Grid

This is the basic structure of a Bootstrap grid.

  • Create a div with class="row".
  • Add the desired number of divs for each column with class = ".col-*-num".
  • All of the numbers the .col-*-num classes for each row should always add up to 12.

This is the basic structure of a Bootstrap grid.

<div class="row">
  <div class="col-*-*"></div>
</div>

Grid System Rules

Some Bootstrap grid system rules:

  • Rows must be placed within a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding.
  • Rows should contain horizontal groups of columns
  • Content should be placed within columns, and only columns may be immediate children of rows
  • Predefined classes like .row and .col-sm-4 are available
  • Columns create gutters between column content via padding. That padding is offset in rows for the first and last column via negative margin on rows
  • Grid columns specify the number of 12 available columns to span. For example, three equal columns would use three .col-sm-4
