Bootstrap Reference - List Groups

List Groups

Basic List Groups

The basic list group is used for dispalying an unordered list with list items. List items may be simple or complex.

To create a basic list group, use an <ul> element with class .list-group, and <li> elements with the .list-group-item class.

Contextual Classes

Contextual classes can be used to color list items.

The classes that can be used are: .list-group-item-success, .list-group-item-info, .list-group-item-warning, and .list-group-item-danger.

Custom Content

List group item can contain complex HTML content.

Bootstrap provides the classes .list-group-item-heading and .list-group-item-text which can be used for formatting.
