Navigation Bar

Needs breakdown of .navbar, .navbar-header, .navbar-brand, and .navbar-nav

Basic Navigation Bar

A standard navigation bar is created using the <nav> element with the .navbar and .navbar-default classes.

The navigation bar can adjust its layout from horizontal to vertical, based on screen size.

Fixed Navigation Bars

Top Fixed Navigation Bar

Adding the .navbar-fixed-top class to a navbar will fix the navbar to the top of the page.

Page content will scroll behind the navbar.

See an example of Fixed Navigation Bars

Bottom Fixed Navigation Bar

Adding the .navbar-fixed-bottom class to a navbar will fix the navbar to the top of the page.

Page content will scroll behind the navbar.

See an example of Fixed Navigation Bars

Customized Navigation Bars

Right-Aligned Navigation

The .navbar-right class is used to right-align .navbar elements.

Separate .navbar elements are used for the left and right nav containers inside a <nav>
