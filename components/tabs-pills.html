Tabs

Tabs are created by adding the .nav and .nav-tabs classes to an unordered list.

The selected tab can be indicated by applying the .active class to one of the <li> elements in the tab set.

Individual tabs can contain dropdown menus.

The .dropdown class is added to the <li> element that represents the tab with the dropdown.

A button or anchor is added inside the .dropdown element, along with a <ul> containing the menu items. (see Dropdown Class for reference.)

Pills

Pills are menu links with shapes around them.

Pills are created by adding the .nav and .nav-pills classes to an unordered list.

The selected pill can be indicated by applying the .active class to one of the <li> elements in the pill set.

Justified Tabs and Pills

Adding the .nav-justifiedclass to the .nav-tabs or .nav-pills class creates a set of tabs or pills that are centered and justified across the screen.

List items will be stacked and centered at smaller resolutions below 768 pixels.

Toggleable / Dynamic Tabs

To make the tabs toggleable, add the data-toggle="tab" attribute to the <a> element in each <li> in the tab set.

Apply the .tab-contentclass to a wrapper <div> containing a series of elements with the .tab-pane class.

Each .tab-pane should have a unique ID.

Adding the .fade class to a .tab-pane makes the tab content fade in and out when a tab is selected.

JS Tab Events

The show.bs.tab event is triggered when the tab is about to be shown.

The shown.bs.tab event is triggered when the tab is fully shown (after CSS transitions have completed).

The hide.bs.tab event is triggered when the tab is about to be hidden.

The hidden.bs.tab event is triggered when the tab is fully hidden (after CSS transitions have completed).
