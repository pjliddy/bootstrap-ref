Bootstrap Reference - Forms

Default Form Settings

Form Layouts

Bootstrap provides three types of form layouts: vertical (default), horizontal, or inline.

Wrap labels and form controls in <div class="form-group">; for correct spacing

Add the .form-control class to all textual <input>, <textarea>, and <select> elements

All <input>, <textarea>, and <select> elements with the.form-control class have a width of 100%.

Always use the correct ARIA role: <form role="form">

Vertical Form

A vertical form is created by the standard <form> element.

Horizontal Form

A horizontal form is created by applying the using the .form-horizontal class to a standard <form> element.

The .control-label class should be applied to all <label> elements to ensure proper positioning next to their corresponding input element.

This horizontal form layout uses a two and ten column layout with the col-sm-offset-2 class applied to the checkbox and button to shift their layout to the right.

Inline Form

An inline form is created by adding the .form-inline class to the <form> element.

All of the elements of an inline form are inline, left-aligned (only on viewports greater than 768 pixels).
