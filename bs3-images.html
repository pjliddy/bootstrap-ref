Responsive Styles

Responsive Image

The .img-responsive class scales the image to the parent element and applies display: block; and max-width: 100%; and height: auto; to the image:

Responsive Embed

The .embed-responsive class is applied to a container to create a responsive video that will scale to the parent element.

The containing <div> defines the aspect ratio of the video and lets videos or slideshows scale properly on any device. Classes can be applied directly to <iframe>, <embed>, <video>, and <object> elements.

The .embed-responsive-item class is applied to the <iframe>

There are two aspect ratio classes that can be applied to the responsive video: .embed-responsive-16by9 and .embed-responsive-4by3
