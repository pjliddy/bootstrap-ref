Text Formatting

Strong (Bold) Element

Use the <strong> element to indicate text that is bold for formatting purposes.

Emphasized (Italicized) Element

Use the <em> element to indicate text that is bold for formatting purposes.

Underlined Element

Use the <u> element to indicate text that is underlined for formatting purposes.

Strikethrough Element

Use the <s> element to indicate text that is no longer relevant for formatting purposes.

Highlight Element

Use the <mark> element to highlight text for formatting purposes.

Deleted Element

Use the <del> element to indicate text that has been deleted for editing purposes.

Inserted Element

Use the <ins> element to indicate text that has been inserted for editing purposes.

Abbreviation Element

The <abbr> element is used to mark up an abbreviation or acronym:

The WHO was founded in 1948.

Initialism Class

The .initialism class displays the text inside an <abbr> element in a slightly smaller font size.

The WHO was founded in 1948. (normal abbr)

The WHO was founded in 1948. (initialism abbr)

Use the <address> element to present contact information:

Code Elements

Keyboard Input Element

To indicate input that is typically entered via the keyboard, use the <kbd> element.

Use ctrl + p to open the Print dialog box.

Code Snippet Element

Inline snippets of code should be embedded in the <code> element.

Sample Report Element

The <samp> tag defines sample output from a computer program. This formatting can be done with CSS as well

Variable Element

Variables should be displayed using in the <var> element: x = ab + y

Lists

Ordered List Element

The <ol> class displays a list of items in numerical order:

  1. Coffee
  2. Tea
    • Black tea
    • Green tea
  3. Milk

Unordered List Element

The <ul> class displays a list of items:

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Unstyled List Class

The .list-unstyled class removes the default list-style and left margin on list items (immediate children only):

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

Inline List Class

The .list-inline class places all list items on a single line:

  • Coffee
  • Tea
  • Milk

Description Lists

Description List Element

The <dl> element indicates a description list. <dt> tag shows terms. <dd> tag shows descriptions.

Coffee
- black hot drink
Milk
- white cold drink

Horizontal Description List Class

The .dl-horizontal class lines up the description list side-by-side when the browser window expands:

Coffee
- black hot drink
Milk
- white cold drink
