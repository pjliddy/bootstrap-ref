Collapse

Basic Collapse

The .collapse class creates a collapsible element containing content that will be shown or hidden when a button or link is clicked.

The data-toggle="collapse" attribute is added to an <a> or a <button> element.

The data-target="#id" attribute also added to trigger the collapsible content element "#id".

Accordion

The .collapse class can extend the panel group component to create an accordion.

The data-parent attribute makes sure that all collapsible elements under the specified parent will be closed when one of the collapsible item is shown.

Bootstrap Collapse JS

Collapse Using JavaScript

Collapsable elements can be controlled manually with a JavaScript calls to:

  • $('.show').collapse() from a link or a button element
  • $('.hide').collapse() from a link or a button element
  • $('.toggle').collapse() from a link or a button element

Collapse Events

The show.bs.collapse event is triggered when the collapsible element is about to be shown.

The shown.bs.collapse event is triggered when the collapsible element is fully shown (after CSS transitions have completed).

The hide.bs.collapse event is triggered when the collapsible element is about to be hidden.

The hidden.bs.collapse event is triggered when the collapsible element is fully hidden (after CSS transitions have completed).
